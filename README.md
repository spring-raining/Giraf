Giraf - Powerful & Free GIF Creator - https://spring-raining.github.io/Giraf

## Usage on local

1.  Install Node.js

2.  Clone Giraf

    ```sh
    $ git clone https://github.com/spring-raining/Giraf
    ```

3.  Build & Run

    ```sh
    $ cd Giraf
    $ npm install
    $ npm start
    ```

4.  Access `http://localohst:8080`
